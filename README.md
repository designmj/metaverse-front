
# 🖥 학습 및 전시 플랫폼 HIP


## 📕 Introduce
- HIP은 SW아카데미 학생들의 학습 및 프로젝트 전시를 위한 플랫폼입니다.
- 홍익대학교 SW아카데미 학생들의 다양한 프로젝트가 전시 되어 있습니다.
- 아카데미 수업을 위한 강좌를 볼 수 있는 학습룸이 있습니다.
